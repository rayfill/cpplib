C++による自作のライブラリです。
一部のOSやデバイスなど外部システムに直接つながるクラスを除いてユニットテストが実装されています。
基本的にヘッダファイル内にすべてを詰め込んだtemplateクラスで構成されています。
またできる限りC++規格上に収まるように作っています。Endian関係などビットレベルで直接扱わないとダメな部分は無理ですが、それ以外であればreinterpret_castをつかっている部分以外はかなりの移植性を持たせたつもりです（実際Makeファイルでは大体のテストがpedanticオプションでコンパイルされるようにしています）。
MPI(多倍長整数)クラスやDIBitmapテンプレートクラス、Blitterクラスなどはテンプレートのインライン展開および最適化を期待して作られているため、最適化レベルを多少は上げて作成しないと、とんでもなく遅くなることがあります。
ただしインライン展開および最適化されればクラスレベルで抽象化している部分がすべて畳み込まれるため、実行速度は結構速い速度で実行されます。

実際のところ、一番時間がかかってるのはCryptography(およびその技術的基底となっている多倍長整数クラス)関連でしょうか。
暗号関連をほとんど一から覚えながら、最適化方法探して実装していったものです。
とりあえずJava実装の数倍程度の速度が出るところまではこぎつけたのですがGNU bnライブラリ（GNUの多倍長整数Cライブラリ）には結構水をあけられています・・・メモリ管理周りでのコスト差があるのと積演算、商演算周りがまだまだあまいところがありますし・・・内部表現を持っているSTL vectorのコストを削らないと追いつけそうに無いですね・・・

各ライブラリの使い方は一部のものに関してはDoxygen用の記述がされているものもありますが、何も書いて無いものも結構あります。

基本的にCygwin上でg++ 3.4.4 (Cygming special) とCppUnit 1.10.2をつかってテストしています。
大体のものに関してはVisualStudio.Net2003上でもビルドなどはした経験があるのでそのままincludeして使うことができます。
Cryptography, Thread(WinThread含む), math, Graphics, Win32などは動作することを確認してあります。
