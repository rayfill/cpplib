C++による自作のライブラリです。

一部のOSやデバイスなど外部システムに直接つながるクラスを除いてユニット
テストが実装されています。基本的にヘッダファイル内にすべてを詰め込んだ
templateクラスで構成されています。またできる限りC++規格上に収まるよう
に作っています。Endian関係などビットレベルで直接扱わないとダメな部分は
無理ですが、それ以外であればreinterpret_castをつかっている部分以外はか
なりの移植性を持たせたつもりです（実際Makeファイルでは大体のテストが
pedanticオプションでコンパイルされるようにしています）。MPI(多倍長整数)
クラスやDIBitmapテンプレートクラス、Blitterクラスなどはテンプレートの
インライン展開および最適化を期待して作られているため、最適化レベルを多
少は上げて作成しないと、とんでもなく遅くなることがあります。ただしイン
ライン展開および最適化されればクラスレベルで抽象化している部分がすべて
畳み込まれるため、実行速度は結構速い速度で実行されます。

実際のところ、一番時間がかかってるのはCryptography(およびその技術的基
底となっている多倍長整数クラス)関連でしょうか。暗号関連をほとんど一か
ら覚えながら、最適化方法探して実装していったものです。とりあえずJava実
装の数倍程度の速度が出るところまではこぎつけたのですがGNU bnライブラリ
（GNUの多倍長整数Cライブラリ）には結構水をあけられています・・・メモリ
管理周りでのコスト差があるのと積演算、商演算周りがまだまだあまいところ
がありますし・・・内部表現を持っているSTL vectorのコストを削らないと追
いつけそうに無いですね・・・

各ライブラリの使い方は一部のものに関してはDoxygen用の記述がされている
ものもありますが、何も書いて無いものも結構あります。

基本的にCygwin上でg++ 3.4.4 (Cygming special) とCppUnit 1.10.2をつかっ
てテストしています。大体のものに関してはVisualStudio.Net2003上でもビル
ドなどはした経験があるのでそのままincludeして使うことができます。
Cryptography, Thread(WinThread含む), math, Graphics, Win32などは動作す
ることを確認してあります。
